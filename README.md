## 🚚 Current Project: Deliver App

I am actively building **Deliver App** — a microservice-based delivery platform with courier tracking, order management, admin dashboards, notifications, location updates, and API Gateway routing.

### Core services

```text
Flutter / Web Client
        ↓
gateway-service :8090
        ↓
auth-service          :8081
order-service         :8082
map-service           :8083
location-service      :8084
notification-service  :8085
organization-service  :8086
```

### Recently implemented in Gateway

- ✅ Single entry point for frontend: `gateway-service :8090`
- ✅ Routing to all backend microservices
- ✅ Blocking external access to `/api/internal/**`
- ✅ `X-Request-Id` filter for request tracing
- ✅ Gateway access logs for monitoring and debugging
- ✅ HTTP client timeout configuration
- ✅ CORS configuration for Flutter/Web clients
